## 📚 Sobre o repositório

Este repositório foi criado com o objetivo de **praticar a linguagem Java utilizando o VSCode e manter a lógica de programação sempre afiada**.

Aqui serão adicionados exercícios, desafios e pequenos projetos desenvolvidos no **Visual Studio Code**, com foco em treinar raciocínio lógico e melhorar a resolução de problemas utilizando Java.

## 🎯 Objetivo

- Manter a lógica de programação sempre em dia  
- Praticar resolução de problemas  
- Reforçar conceitos fundamentais da programação  
- Evoluir constantemente no uso da linguagem Java  
- Utilizar o **VSCode como ambiente de desenvolvimento**

## 🧠 Motivação

A prática constante é essencial para evoluir na programação.  
Este repositório serve como um espaço para **treinar lógica, testar soluções e acompanhar a evolução no aprendizado de Java utilizando o VSCode**.
